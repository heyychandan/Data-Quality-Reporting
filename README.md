# Data-Quality-Reporting
Hosts the Data Quality and Persisted Reporting Service for ICAI member data, a robust application designed to automatically validate incoming data streams, calculate quality metrics, and persist historical reports for longitudinal monitoring and auditing. It ensures data reliability across pipelines by providing an interactive dashboard.

The core Colab/Jupyter Notebook containing all Python code, data loading, preprocessing, feature engineering, and the final risk scoring logic.
Model Name: icai_risk_model_v1.h5
